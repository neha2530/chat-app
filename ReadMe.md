Steps to run the application:

Install Dependencies in frontend
1> cd ./frontend/chat-ui
2> npm install

Go back to Root
3> cd ..

Install Dependencies in Backend
4> cd ./backend 
5> npm install

Go Back to Root
6> cd ..

Start Frontend
7> cd ./frontend/chat-ui
8> npm start




Open new terminal and go to application Root and start backend

1> cd backend
2> npm start
